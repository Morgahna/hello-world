A Program to Greet the World

This is an attempt to greet everyone on Earth using various different human languages. We're interested to see exactly how long this list can become.

We'll start with languages that Morgahna is could offer without using Google

  • Hello
  • G'Day
  • Bonjour
  • Kia Ora
  • Ola
  • Mwirwe, isi!
  • Guten tag, Welt!
